# COMP472-Project
 In this project, your task is to perform image classification on the CIFAR-10 dataset using some of the AI
 models learned in this class. You will first analyze the CIFAR-10 dataset, and pre-process the images. You will
 then build four AI models, Naive Bayes, a decision tree, a multi-layer perception, and a convolutional neural
 network (CNN) using Python and Pytorch to detect the 10 object classes in the CIFAR-10 dataset. You will
 then apply basic evaluation metrics: accuracy, confusion matrix, precision, and recall.
